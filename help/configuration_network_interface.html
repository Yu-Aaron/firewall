接口

接口清单用于对整个防火墙子物理接口，子接口，VAAN，桥各类接口进行查看和管理。

 

•子接口配置

子接口是通过协议和技术将一个物理接口虚拟出来的多个逻辑接口。通过子接口，可以将一个路由器的单个物理接口划分为多个子接口，从而实现多个VLAN间的路由和通信。

•VLAN接口配置

VLAN接口可以将一组逻辑上的设备和用户，这些设备和用户并不受物理位置的限制，根据功能、部门及应用等因素将它们组织起来，相互之间的通信就好像它们在同一个网段中一样。

•桥接口配置

桥接口可以以网桥的方式连接两端LAN。
